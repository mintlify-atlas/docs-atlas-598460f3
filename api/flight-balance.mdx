## Overview

The Flight Balance API manages weight and balance calculations for aircraft. These calculations are critical for flight safety, ensuring the aircraft operates within structural limits and the center of gravity is within the permitted range.

<Note>
All endpoints require JWT authentication via cookie and appropriate permissions:
- Read operations require `despacho:read` permission
- Write operations require `despacho:write` permission
</Note>

## Create Flight Balance

<Api method="POST" path="/flight-balances" />

Creates a new weight and balance calculation for an aircraft. Includes all subtotals, totals, and detail items (rows in the W&B table). The registering user is automatically obtained from the JWT token.

### Request Body

<ParamField body="subtotal_operating_empty_weight" type="number" required>
  Operating empty weight in pounds
</ParamField>

<ParamField body="subtotal_operating_empty_cg" type="number" required>
  Operating empty center of gravity
</ParamField>

<ParamField body="subtotal_operating_empty_moment" type="number" required>
  Operating empty moment (weight x arm)
</ParamField>

<ParamField body="subtotal_operating_fuel_weight" type="number" required>
  Operating weight with fuel in pounds
</ParamField>

<ParamField body="subtotal_operating_fuel_cg" type="number" required>
  Center of gravity with fuel
</ParamField>

<ParamField body="subtotal_operating_fuel_moment" type="number" required>
  Moment with fuel
</ParamField>

<ParamField body="subtotal_operating_fuel_mac" type="number" required>
  Mean Aerodynamic Chord (MAC) with fuel in percentage
</ParamField>

<ParamField body="total_take_off_weight" type="number" required>
  Total takeoff weight in pounds
</ParamField>

<ParamField body="total_take_off_cg" type="number" required>
  Center of gravity at takeoff
</ParamField>

<ParamField body="total_take_off_moment" type="number" required>
  Total moment at takeoff
</ParamField>

<ParamField body="total_take_off_mac" type="number" required>
  MAC at takeoff in percentage
</ParamField>

<ParamField body="total_landing_weight" type="number" required>
  Total landing weight in pounds
</ParamField>

<ParamField body="total_landing_cg" type="number" required>
  Center of gravity at landing
</ParamField>

<ParamField body="total_landing_moment" type="number" required>
  Total moment at landing
</ParamField>

<ParamField body="total_landing_mac" type="number" required>
  MAC at landing in percentage
</ParamField>

<ParamField body="aircraft_id" type="integer" required>
  ID of the aircraft
</ParamField>

<ParamField body="pilot_command_id" type="integer" required>
  ID of the pilot in command
</ParamField>

<ParamField body="pilot_officer_id" type="integer">
  ID of the first officer (optional)
</ParamField>

<ParamField body="details" type="array" required>
  Array of detail items for the calculation
  
  <Expandable title="Detail object properties">
    <ParamField body="item" type="string" required>
      Item description (e.g., "Piloto Cmte.", "Pasajero 1")
    </ParamField>
    
    <ParamField body="weight" type="number" required>
      Weight of the item in pounds
    </ParamField>
    
    <ParamField body="cg" type="number" required>
      Center of gravity position (arm) in inches
    </ParamField>
    
    <ParamField body="moment" type="number" required>
      Calculated moment (weight x arm)
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="flightBalance" type="object">
  The created flight balance object with all fields and relationships
  
  <Expandable title="Flight balance properties">
    <ResponseField name="id" type="integer">
      Unique identifier for the calculation
    </ResponseField>
    
    <ResponseField name="subtotal_operating_empty_weight" type="number">
      Operating empty weight in pounds
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Creation timestamp (ISO 8601)

      Last update timestamp (ISO 8601)
    </ResponseField>
    
    <ResponseField name="details" type="array">
      Array of detail items included in the calculation
    </ResponseField>
  </Expandable>
</ResponseField>

## List Flight Balances

Retrieves a paginated list of flight balance calculations with optional filters.

<ParamField query="page" type="integer" default="1">
  Page number (minimum: 1)
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Records per page (minimum: 1, maximum: 100)
</ParamField>

<ParamField query="aircraft_id" type="integer">
  Filter by aircraft ID
</ParamField>

<ParamField query="registered_by_id" type="integer">
  Filter by registering user ID
</ParamField>

<ParamField query="pilot_command_id" type="integer">

</ParamField>

<ParamField query="date_from" type="string">
  Filter from date (format: YYYY-MM-DD)
</ParamField>

## Update Flight Balance

<Api method="PATCH" path="/flight-balances/{id}" />

Partially updates an existing flight balance calculation. Implements PATCH semantics: only sent fields are updated. If details are sent, they replace all existing details.

### Path Parameters

<ParamField path="id" type="integer" required>
  ID of the calculation to update
</ParamField>

### Request Body

All fields from the create endpoint are optional. Only include fields you want to update.

<ParamField body="subtotal_operating_empty_weight" type="number">
  Operating empty weight in pounds
</ParamField>

<ParamField body="details" type="array">
  If provided, replaces all existing details
</ParamField>

<Info>
For a complete list of updatable fields, see the [Create Flight Balance](#create-flight-balance) section.
</Info>

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="flightBalance" type="object">
  The updated flight balance object
</ResponseField>

<RequestExample>
```bash cURL
curl -X PATCH https://api.sigsaho.com/flight-balances/123 \
  -H "Content-Type: application/json" \
  -H "Cookie: token=your-jwt-token" \
  -d '{
    "total_take_off_weight": 3550,
    "total_landing_weight": 3250
  }'
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "message": "Calculo de peso y balance actualizado exitosamente",
  "flightBalance": {
    "id": 123,
    "total_take_off_weight": 3550,
    "total_landing_weight": 3250,
    "updated_at": "2026-03-04T11:00:00.000Z"
  }
}
```

```json 404 Not Found
{
  "error": "Calculo de peso y balance no encontrado"
}
```
</ResponseExample>
