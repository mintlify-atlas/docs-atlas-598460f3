## Overview

The Reports API generates PDF documents for various operational entities. All endpoints return PDF buffers with appropriate headers for download or display.

<Note>
All endpoints require JWT authentication via cookie and `reports:read` permission.
</Note>

## Generate Transfer Card

<Api method="GET" path="/reports/transfers/{id}/card" />

Generates an informative PDF card for a transfer operation.

### Path Parameters

<ParamField path="id" type="integer" required>
  ID of the transfer
</ParamField>

### Response

Returns a PDF buffer with content type `application/pdf` and disposition header set to download the file as `tarjeta-traslado-{id}.pdf`.

<ResponseField name="Content-Type" type="header">
  `application/pdf`
</ResponseField>

<ResponseField name="Content-Disposition" type="header">
  `attachment; filename=tarjeta-traslado-{id}.pdf`
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.sigsaho.com/reports/transfers/123/card \
  -H "Cookie: token=your-jwt-token" \
  --output tarjeta-traslado-123.pdf
```

```javascript JavaScript
fetch('https://api.sigsaho.com/reports/transfers/123/card', {
  method: 'GET',
  credentials: 'include',
  headers: {
    'Cookie': 'token=your-jwt-token'
  }
})
.then(response => response.blob())
.then(blob => {
  const url = window.URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = 'tarjeta-traslado-123.pdf';
  a.click();
});
```
</RequestExample>

<ResponseExample>
```json 200 OK
[Binary PDF data]
```

```json 400 Bad Request
{
  "error": "ID de traslado inválido"
}
```

```json 404 Not Found
{
  "error": "Traslado no encontrado"
}
```

```json 500 Internal Server Error
{
  "error": "Error al generar el reporte"
}
```
</ResponseExample>

<Api method="GET" path="/reports/users/{id}/card" />

Generates a technical personnel file (expediente) in PDF format, including personal data, documents, and training records.

### Path Parameters

<ParamField path="id" type="integer" required>
  ID of the user
</ParamField>

### Response

Returns a PDF buffer with content type `application/pdf` and disposition header set to download the file as `expediente-personal-{id}.pdf`.

<ResponseField name="Content-Type" type="header">
  `application/pdf`
</ResponseField>

<ResponseField name="Content-Disposition" type="header">
  `attachment; filename=expediente-personal-{id}.pdf`
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.sigsaho.com/reports/users/789/card \
  -H "Cookie: token=your-jwt-token" \
  --output expediente-personal-789.pdf
```

```go Go
package main

import (
    "fmt"
    "io"
    "net/http"
    "os"
)

func main() {
    req, _ := http.NewRequest("GET", 
        "https://api.sigsaho.com/reports/users/789/card", nil)
    req.Header.Add("Cookie", "token=your-jwt-token")
    
    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()
    
    out, _ := os.Create("expediente-personal-789.pdf")
    defer out.Close()
    
    io.Copy(out, resp.Body)
    fmt.Println("PDF downloaded successfully")
}
```
</RequestExample>

## Generate Aircraft Monthly Report

<Api method="GET" path="/reports/aircraft/{id}/card" />

Generates a monthly status report for an aircraft in PDF format, including general information, operational data for the current month, maintenance and inspections, and critical components.

### Path Parameters

<ParamField path="id" type="integer" required>
  ID of the aircraft
</ParamField>

### Response

Returns a PDF buffer with content type `application/pdf` and disposition header set to download the file as `reporte-aeronave-{id}.pdf`.

<ResponseField name="Content-Type" type="header">
  `application/pdf`
</ResponseField>

<ResponseField name="Content-Disposition" type="header">
  `attachment; filename=reporte-aeronave-{id}.pdf`
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.sigsaho.com/reports/aircraft/12/card \
  -H "Cookie: token=your-jwt-token" \
  --output reporte-aeronave-12.pdf
```

```php PHP
<?php
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, 
    'https://api.sigsaho.com/reports/aircraft/12/card');
curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
curl_setopt($ch, CURLOPT_COOKIE, 'token=your-jwt-token');

$result = curl_exec($ch);
if (curl_getinfo($ch, CURLINFO_HTTP_CODE) === 200) {
    file_put_contents('reporte-aeronave-12.pdf', $result);
    echo 'PDF downloaded successfully';
}
curl_close($ch);
?>
```
</RequestExample>

## Best Practices

<CardGroup cols={2}>
  <Card title="Error Handling" icon="shield-check">
    Always check HTTP status codes before processing the PDF. Handle 404 errors gracefully when entities don't exist.
  </Card>
  
  <Card title="File Naming" icon="file-signature">
    Use the filename from the `Content-Disposition` header or construct meaningful names using entity IDs and dates.
  </Card>
  
  <Card title="Caching" icon="database">
    Consider caching frequently accessed reports with appropriate TTL to reduce server load.
  </Card>
  
  <Card title="Large Files" icon="file-arrow-down">
    Implement streaming or chunked downloads for better performance with large PDF files.
  </Card>
</CardGroup>

<Tip>
For batch report generation, implement rate limiting and queuing to prevent server overload.
</Tip>
